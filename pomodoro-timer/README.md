# Pomodoro Timer

Offline work/break timer with persistent settings.

## Features
- Start, pause, reset controls.
- Work, short-break, and long-break cycles.
- Configurable long-break interval.
- LocalStorage persistence for timing settings.

## Permissions
- Notifications are optional and may be denied.
- Audio alert attempts to play via Web Audio API; browsers may restrict autoplay until user interaction.

## Notes
- Browser background throttling can introduce timing drift in inactive tabs.
