# Markdown Preview

Offline split-pane markdown editor with sanitized rendering.

## Features
- Live editor and preview.
- Basic markdown support for headings, emphasis, code, links, and simple list lines.
- Clear, copy-preview-HTML, and download-markdown actions.
- Local persistence with localStorage.

## Security Notes
- Raw HTML in input is escaped before rendering.
- Links open in new tab with `noopener noreferrer`.
